## 1. 项目概述

### 1.1 产品定位

**心之清文旅** 是一个以「心灵成长」为主题的文旅咨询小程序。用户通过小程序浏览心灵培训、疗愈、保养、旅游四条业务线，提交成长定制需求，由顾问一对一承接转化。

品牌核心语：**入道调心境** — 在星空与自然之间，回到更清楚、更纯粹的自己。

### 1.2 技术架构

| 层 | 技术 |
|---|------|
| 前端 | 微信小程序原生开发（Skyline 渲染引擎 + glass-easel 组件框架） |
| 后端 | 待接入微信云开发 / 云数据库 / 云存储 |
| 数据 | 当前体验版使用前端内置 mock + Storage 持久化报名需求 |
| 降级 | 不请求本地服务器，体验版可离线展示和提交本地报名记录 |
| 自定义组件 | `custom-tab-bar`（全局底部导航） |

### 1.3 当前阶段

- UI 已完成并进入上线功能补全阶段
- 表单提交走通（当前写入小程序 Storage，后续切换云数据库）
- 暂未接微信登录、支付与真实订单流转

## 3. 页面详细规格

### 3.1 首页 (`home`)

2. **Hero 卡片**（渐变蓝背景 + 背景图）
   - 顶部徽章："Global Mind Growth Journey"
   - 英文大标题（衬线字体，68rpx）：
     ```
     Xinzhiqing
     Cultural Tourism
     ```
   - 中文标题："心 之 清 文 旅 ・ 入 道 调 心 境"
   - 中文副文案 + 英文副文案
   - 两个按钮：`开启体验`（primary 金色实心）、`了解更多`（secondary 白色描边）
   - 右下方装饰性半透明圆形光斑

3. **四格快捷入口**（grid 4 列）
   - 每格：圆形图标 + 文字标签
   - 数据项：`{ icon, title }`
   - 当前内容：心灵培训、心灵保养、心灵疗愈、心灵旅游

4. **成长体系区**
   - Section 标题："成长体系" + "4 LEVELS"
   - Chip 列表：初级 / 中级 / 高级 / 使命（首个高亮）

5. **精选路线区**
   - Section 标题："精选路线" + "FEATURED"
   - 两列网格卡片：`{ title, subtitle, desc }`
   - 点击进入路线详情页

### 3.2 成长定制 (`growth`)

**数据来源：** 表单本地状态 + Storage 持久化，后续切换云数据库

**功能流程：**
1. 顶部品牌栏（图标"定"，标题"成长定制"）
2. Hero 卡片（描述成长定制的价值主张）
3. **分类 Chip** — 4 个选项：培训 / 游学 / 线路 / 定制，点击切换，联动表单 `serviceType` 字段
4. **需求采集表单**（玻璃态卡片）
   - 字段（3 行双列 + 1 个文本域）：

     | 字段 | 占位符 | key |
     |------|--------|-----|
     | 出发城市 | "出发城市 / Departure" | `city` |
     | 目的地方向 | "目的地方向 / Destination" | `destination` |
     | 人数 | "人数 / Group Size" | `groupSize` |
     | 时间 | "时间 / Time Window" | `timeWindow` |
     | 预算 | "预算 / Budget" | `budget` |
     | 类型 | "类型 / Service Type" | `serviceType` |
     | 备注 | "简单描述你想参加的培训主题…" | `note` |

   - 所有字段必填，任一为空 `wx.showToast` 提示
5. 两个按钮：`提交顾问`（调用 API）、`保存`（写入 Storage 草稿）
6. **AI 互动预留区**（soft 卡片）— 占位说明

**交互细节：**
- 草稿自动保存到 `wx.getStorageSync('growth_request_draft')`
- 页面 onShow 时恢复草稿
- 如果从路线详情页跳转过来，自动预填路线名称到备注字段（`growth_request_prefill`）
- 提交成功后清除草稿，写入 `latest_growth_request`，跳转提交成功页
- 提交中按钮显示 loading，不可重复点击

## 7. 业务逻辑

### 7.1 四级成长体系

```
初级 → 中级 → 高级 → 使命
```

从自我意识建立 → 系统维护 → 更高层次整合与传播。

### 7.2 四大业务线

| 快捷入口 | 对应方向 |
|---------|---------|
| 心灵培训 | 建立主线、系统化提升 |
| 心灵保养 | 长期维护、清净松弛 |
| 心灵疗愈 | 情绪释放、恢复平衡 |
| 心灵旅游 | 个性化定制、体验优先 |

### 7.3 顾问承接模式

- 所有需求提交后状态为「已提交，待顾问跟进」
- 顾问通过联系页展示（微信 + 电话）
- 需求列表和咨询记录用于追踪承接链路
- 非自助下单，体验设计围绕「提交需求 → 顾问联系」展开

### 7.4 表单草稿与预填

- **草稿保存：** `wx.setStorageSync('growth_request_draft', form)`，onShow 自动恢复
- **路线预填：** 路线详情页点击「提交咨询」时，将路线名写入 `growth_request_prefill`，成长定制页 onShow 检测到后自动追加到备注
- **提交后清理：** 成功后删除 draft key

## 8. 非功能特性

### 8.1 容错设计

- 当前体验版不依赖本地服务器，可直接在手机体验版运行
- 报名需求先写入 Storage，保证无云端时也能走完整流程
- 后续接云数据库后，需保留提交失败 toast 与重试机制

### 8.2 性能

- 使用 Skyline 渲染引擎（新一代小程序渲染）
- 首页通过 `/api/bootstrap` 单次请求获取全量首页数据，减少请求数
- 草稿使用 Storage 本地同步，无需网络

### 8.3 空状态覆盖

所有列表页均实现空状态 UI：
- 需求列表：引导去成长定制页填写
