⚑ 设计决策记录 (v0)

  • 配色: 深空墨蓝 #0d1f3c + 品牌蓝 #124a7c + 暖金 #d4a12c + 自然绿 #7a9e7e。原版大面积蓝色改为深空底 + 暖金点缀。
  • Hero 背景: 用 CSS 径向渐变画星光点阵替代外部图片 URL，呼应「星空/宇宙」品牌意象。
  • 字体: 英文标题用 Georgia 衬线体（替代 Times New Roman），中文保持 PingFang SC。标题字号由 68rpx 缩小到 38px 更精致。
  • 快捷入口: 单字图标 → 几何符号 + 独立配色圆形底。颜色区分四条业务线。
  • 成长体系: Chip → 阶梯式分段按钮，用罗马数字 Ⅰ-Ⅳ 增加仪式感。
  • 路线卡片: 增加顶部色条（wave）提升辨识度，四种颜色对应四个方向。
  • 页面底色: 从冷白渐变改为暖白 #f8f6f1，传递温度感。
  • 按钮: 主按钮从蓝色改为暖金色，更突出 CTA。
