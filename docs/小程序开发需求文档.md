# 心之清文旅小程序开发需求文档

版本：v1.0  
日期：2026-06-23  
对应原型：`docs/心之清文旅小程序UI线稿-领导预览.html`

## 1. 项目目标

基于当前确认的 UI 提案，搭建一套可在微信开发者工具中直接运行的微信小程序骨架，用于：

1. 还原核心页面结构与视觉层级
2. 作为后续联调、接接口、接真实内容的基础工程
3. 给领导和设计继续确认方向

## 2. 底部导航

固定四栏：

1. 首页
2. 成长定制
3. 资讯
4. 我的

采用自定义 `tabBar`，风格参考最终 UI 提案。

## 3. 页面范围

### 3.1 首页

包含：

- 自定义顶部品牌栏
- Hero Banner
- 四类核心业务入口
- 成长体系层级
- 精选路线模块

### 3.2 成长定制

包含：

- 顶部品牌栏
- Hero Banner
- 成长类别筛选
- 需求采集表单
- 说明区

### 3.3 资讯

包含：

- 顶部品牌栏
- Hero Banner
- 分类 chips
- 列表卡片
- 价值观模块

### 3.4 我的

包含：

- 顶部品牌栏

- 统计模块
- 菜单列表

### 3.5 路线详情页

包含：

### 3.7 资讯详情页

包含：

- 文章标题

### 3.8 订单详情页

包含：

## 4. 技术约束

### 当前阶段

- 先做静态骨架
- 不接接口
- 不做真实登录
- 不做真实支付
- 不做真实订单流转

### 下一阶段

可逐步接入：

- 真实咨询表单提交
- 内容接口
- 登录态
- 订单接口
- AI 问答能力

## 5. 设计要求

- 尽量 1:1 还原当前最终 UI 提案的结构和层级
- 保持鲜明品牌蓝
- Banner 保持宇宙 / 星空 / 心灵成长方向
- 底部导航图标使用当前统一图标语言
