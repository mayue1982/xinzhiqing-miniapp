# 心之清文旅小程序上线部署清单

## 当前状态

- UI 已通过确认。
- 小程序不再依赖本地 `127.0.0.1:3001` 服务。
- 报名需求当前写入小程序本地 Storage，体验版可完整提交和查看。
- 分享卡片已使用心之清 logo。

## 下一步需要接入的云能力

### 1. 云环境

需要确认：

- 微信云开发环境 ID
- 是否使用微信云开发数据库
- 是否使用微信云存储保存图片
- 是否需要云函数发送顾问通知

### 2. 云数据库集合建议

| 集合 | 用途 |
|------|------|
| `routes` | 精品路线、即将启程路线 |
| `articles` | 资讯文章 |
| `requests` | 用户报名 / 成长定制需求 |
| `orders` | 订单信息 |
| `advisors` | 顾问信息 |
| `settings` | 首页配置、分享配置、品牌配置 |

### 3. 云存储建议目录

| 目录 | 用途 |
|------|------|
| `brand/` | logo、分享图、品牌素材 |
| `home/` | 首页轮播图 |
| `routes/` | 路线图片 |
| `articles/` | 资讯封面图 |
| `uploads/requests/` | 用户报名相关图片 |

## 功能补全优先级

### P0 上线必需

- 报名表单写入云数据库 `requests`
- 管理端可查看报名需求
- 资讯和路线从云数据库读取
- 图片从云存储读取
- 体验版不再出现 localhost 报错

### P1 运营必需

- 管理端新增 / 编辑路线
- 管理端新增 / 编辑资讯
- 管理端上传图片到云存储
- 报名状态管理：待跟进 / 已联系 / 已成单 / 已关闭
- 我的页面读取当前用户提交记录

### P2 后续增强

- 微信登录绑定用户身份
- 订阅消息提醒
- 收藏路线
- 支付 / 订单闭环
- 顾问分配

## 管理端建议

先做轻量管理端，不急着做复杂后台：

- 路线管理
- 资讯管理
- 报名需求管理
- 图片上传
- 顾问联系信息配置

可以优先做成网页管理端，后续再考虑小程序内管理端。

## 需要你提供的信息

1. 微信云开发环境 ID
2. 云存储是否已经开通，以及是否能在微信开发者工具里看到环境
3. 是否要我优先做网页管理端，还是小程序内管理端
4. 报名后是否需要通知到微信、短信、邮箱，还是先只进后台列表
